Wiii visual lab

SVG and Canvas render the same pendulum very differently.

This page keeps one shared pendulum state and draws it twice: once as SVG, once as Canvas. Drag the bob, pause the motion, and change the parameters. The goal is to help us decide where Wiii should use SVG-first figures and where Canvas is still the better tool.

SVG Best for diagrams, charts, arrows, labels, and explanatory inline figures.
Canvas Best for continuous redraw, physics loops, particles, and dense motion.
Wiii hint Article visuals should lean SVG-first. Simulations should stay Canvas or app runtime.
